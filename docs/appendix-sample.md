# 부록: 샘플 파일

> 이 가이드에서 사용한 샘플 파일을 다운로드할 수 있습니다.

## 샘플 원고 (DOCX)

**파일명**: `나도 책을 쓸 수 있을까 - 샘플원고.docx`

이 가이드의 모든 테스트는 이 샘플 원고로 진행되었습니다.

### 포함된 구조

```
표지 (이미지)
판권 (안내 박스 포함)
목차
프롤로그 - 당신도 작가가 될 수 있습니다
Chapter 1 - 왜 지금, 책을 써야 할까
  ├ 당신의 경험은 누군가의 교과서다
  └ 전자책은 가장 빠른 출판 방법이다
Chapter 2 - 7주 만에 책 한 권 쓰는 법
  ├ 기획이 80%다
  └ 하루 30분, 매일 쓰기
에필로그 - 당신의 책이 세상에 나오는 날
저자 소개
```

### 포함된 디자인 요소

| 요소 | 구현 방법 | 변환 결과 |
|------|----------|---------|
| 제목 1/2 스타일 | 워드 Heading 1/2 | ✅ 목차 자동 생성 |
| 페이지 나누기 | Ctrl+Enter | ✅ 챕터 분리 |
| 팁 박스 (💡) | 1x1 테이블 + 배경색 | ✅ 배경색/테두리 보존 |
| 핵심 포인트 박스 (✅) | 1x1 테이블 + 배경색 | ✅ 배경색/테두리 보존 |
| 주의 박스 (⚠️) | 1x1 테이블 + 배경색 | ✅ 배경색/테두리 보존 |
| 인용문 | 1x1 테이블 + 왼쪽 테두리 | ✅ 인용 스타일 보존 |
| 체크리스트 | 1x1 테이블 + ☐ 문자 | ✅ 특수문자 보존 |
| 비교 테이블 | 3열 테이블 + 헤더 배경 | ✅ 복잡 테이블 보존 |
| 통계 카드 | 3열 테이블 + 큰 숫자 | ✅ 레이아웃 보존 |
| 강조 배너 | 1x1 테이블 + 진한 배경 + 흰 글씨 | ✅ 반전색 보존 |
| 이미지 | 본문 삽화 | ✅ 이미지 변환 |
| 볼드/이탤릭 | 직접 서식 | ✅ 기본 서식 보존 |
| 글자 색상 | 직접 서식 | ✅ 색상 보존 |
