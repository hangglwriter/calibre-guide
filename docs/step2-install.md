# 2단계. 캘리버 프로그램 설치

> 무료 프로그램 하나만 설치하면 됩니다.

## 다운로드

**공식 사이트**: [https://calibre-ebook.com/download](https://calibre-ebook.com/download)

Windows, Mac, Linux 모두 지원합니다.

## 설치 방법

1. 위 링크에서 본인 운영체제에 맞는 버전 다운로드 (Windows 64bit 권장)
2. 설치 파일 실행
3. 기본 설정 그대로 **"다음"** 클릭하면 완료
4. 캘리버 실행

<div class="tip-box">
💡 <b>참고</b><br>
설치 중 "캘리버 라이브러리 위치"를 묻는 화면이 나오면 기본값(문서 폴더)으로 두면 됩니다.<br>
나중에 변경할 수 있습니다.
</div>

## 설치 후 첫 화면

캘리버를 처음 실행하면 라이브러리가 비어있는 메인 화면이 나옵니다.
