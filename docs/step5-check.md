# 5단계. 뷰어 확인 + 수정

> 서점에 등록하기 전에 반드시 뷰어에서 확인하세요.

## 5-1. 뷰어에서 확인

EPUB 파일을 아래 뷰어에서 열어서 확인합니다.

| 뷰어 | 방법 |
|------|------|
| **캘리버 뷰어** | 캘리버에서 책 더블클릭 |
| **교보문고 뷰어** | [교보 PC뷰어](https://viewer.kyobobook.co.kr) 설치 후 EPUB 열기 |
| **리디북스 뷰어** | 리디 앱에서 EPUB 파일 열기 |

## 5-2. 확인 체크리스트

- [ ] **표지**가 잘 보이는가
- [ ] **목차**가 제대로 잡혔는가
- [ ] 챕터별로 **페이지가 나뉘는가**
- [ ] **박스/테이블**이 깨지지 않았는가
- [ ] **이미지**가 잘 보이는가
- [ ] **줄간격**이 적당한가 (너무 빽빽하지 않은가)
- [ ] **제목과 본문 사이** 간격이 있는가
- [ ] **볼드/이탤릭/색상**이 유지되는가

## 5-3. 수정이 필요하면

문제를 발견하면 **워드 파일을 수정**하고 다시 변환합니다.

```
워드 수정 → 캘리버 재변환 → 오류 수정 → 뷰어 확인
```

### 자주 나오는 문제와 해결법

| 문제 | 원인 | 해결 |
|------|------|------|
| 목차가 안 잡힘 | 제목 스타일 미적용 | 워드에서 "제목 1/2" 스타일 적용 |
| 빈 줄이 너무 많음 | Enter로 페이지 넘김 | 페이지 나누기(Ctrl+Enter) 사용 |
| 글씨가 빽빽함 | 줄간격 부족 | 캘리버 최소 줄 높이 150% 설정 |
| 이미지 안 보임 | 이미지 링크 깨짐 | 워드에서 이미지 다시 삽입 |
| 박스 배경색 없음 | 테이블 아닌 직접 서식 사용 | 1x1 테이블로 다시 만들기 |

## 5-4. 서점 등록

최종 확인이 끝났으면 서점에 등록합니다.

### 유페이퍼(uPaper) - 추천

- [https://www.upaper.net](https://www.upaper.net)
- EPUB 검사가 가장 엄격 → 여기서 통과하면 다른 서점도 대부분 통과
- 교보문고, 리디북스, 밀리의서재 등에 동시 유통 가능

### 등록 시 참고

- **EPUB 버전**: EPUB 2 (한국 서점 호환성 최고)
- **파일명**: 책 제목으로 (교보에서 파일명이 제목으로 표시됨)
- **표지**: 서점별로 별도 표지 이미지(JPG)를 요구할 수 있음
